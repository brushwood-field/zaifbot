Welcome to Zaifbot!
===================


zaifに対して自動的に取引を行うbot用のライブラリになります。

ご利用は自己責任でお願いします。

本モジュールはテックビューロ非公式です。


----------


使い方(こうなる予定)
-------------
仮想環境を作成しアクティベートした後、下記コマンドを実行してください

> pip install zaifbot  
> init_database

その他、詳細は[wiki][1]をご覧ください

  [1]: https://github.com/techbureau/zaifbot/wiki
